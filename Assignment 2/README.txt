How to Run:
I have used PyCharm IDE to develop my assignment.
There is one python package including 3 files. The files are main.py, featureDetector.py and _init_.py which is empty. 
To Run this assignmen, make one empty project in pycharm and paste the whole python package into it with _init_.py file. Then run main.py file.
I have used 2 images in main.py which are Yosemite1.jpg and Yosemite1.jpg . You can replace with another files also.

Result:
Result Image will displayed with combination of 2 Yosemite images with their matching key points.
Improved result image is also given with another image with improved matching of features between 2 images.
